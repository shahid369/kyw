## 📌 App Overview

**KYW** ("Know Your Cycle") is a period tracker built **for men** — specifically boyfriends and husbands tracking their partner's cycle to be more understanding and supportive.

**Unique positioning:** Every existing app (Flo, Clue, Cycles) is built FOR the woman. KYW is the only app built FOR the man.

**Privacy-first:** We do not collect the partner's personal data. The man enters what he observes. Zero health data mining.

## 🏠 Home Screen Widget (Pro Only)

> **Gating:** Widget setup screen is locked behind Pro. If the user is not Pro, tapping "Add Widget" shows the paywall.

### Why This Matters
- Visible **even when the app is closed** — daily passive value
- Every time he looks at his phone home screen he sees her phase
- Strong **conversion tool** — free users see widget screenshots in the App Store and want it

---

### Platform Implementation

#### Flutter Package
```yaml
# pubspec.yaml
dependencies:
  home_widget: ^0.6.0        # Cross-platform widget support
  workmanager: ^0.5.2        # Background updates
```

#### Android (Glance API / App Widget)
- File location: `android/app/src/main/res/layout/kyw_widget_*.xml`
- Update frequency: Every 24 hours via `WorkManager`
- Widget provider class: `KywWidgetProvider.kt`
- **V1 ships:** Small (2×2) + Medium (4×2)

#### iOS (WidgetKit — Swift)
- File location: `ios/KywWidget/KywWidget.swift`
- Uses **App Groups** to share data between Flutter app and widget
- Update frequency: On app open + midnight background refresh
- **V1 ships:** Small + Medium only *(Lock Screen deferred to v2)*

---

### Data Flow (Privacy-safe)

```
Flutter App  →  SharedPreferences/UserDefaults  →  Widget
              (local only, no network call from widget)
```

The widget **reads cached data** written by the app the last time it was opened.  
No Supabase calls from the widget itself — fully offline, fully private.

**Data written to shared storage by the app:**
```json
{
  "phase": "Luteal",
  "day_in_cycle": 22,
  "days_until_period": 6,
  "cycle_progress": 0.78,
  "partner_name": "Sarah",
  "show_name_on_widget": false,
  "todays_tip": "Plan a quiet evening tonight",
  "phase_color": "#A78BFA",
  "phase_emoji": "🌸"
}
```
